# 阴阳师斗技自动挂机

## 功能说明

根据实际斗技流程设计，自动完成以下操作：

1. **匹配界面** → 自动点击右下角 "战" 按钮
2. **选人/禁选界面** → 自动点击右下角 "确定" 按钮（支持多次点击，应对多轮选人）
3. **战斗界面** → 自动点击左下角 "手动" 按钮（开启自动作战）
4. **结算界面** → 自动点击屏幕中央跳过结算（胜利/失败均适用）
5. 循环进行下一轮斗技

### 新增功能

- **全局热键**：按 F6 一键启停，不用切出游戏
- **战斗统计**：实时显示胜场、负场、胜率、挂机时长
- **定场/定时停止**：设置最多场次或最长时间，达到后自动停止
- **系统托盘**：关闭窗口自动缩到右下角托盘，更隐蔽
- **防检测优化**：贝塞尔曲线鼠标移动、随机点击位置/时长/间隔
- **多阵容管理**：支持多套式神阵容，可编辑、新增、删除
- **自动选人**：进入选人界面后自动搜索并上滑选择预设式神和阴阳师
- **阵容轮换**：每N场自动换阵容，或战败时自动换阵容（防针对）

## 使用方法

### 第一步：制作模板图片

这是最关键的一步！你需要在游戏运行时截取模板图片：

**1. battle_button.png** - 匹配界面的"战"按钮
- 进入斗技匹配界面
- 截取右下角的圆形"战"按钮

**2. confirm.png** - 选人界面的"确定"按钮
- 点击"战"进入匹配后，进入选人/禁选界面
- 截取右下角的鼓形"确定"按钮

**3. manual.png** - 战斗界面的"手动"按钮
- 进入战斗后
- 截取左下角的"手动"按钮

**4. search_button.png** - 选人界面的放大镜搜索按钮（自动选人用）
- 进入选人界面
- 截取左下角的放大镜搜索图标

> **重要**：截图时请使用系统截图工具（如Win+Shift+S或Snipaste），截图分辨率需要与游戏实际运行分辨率一致。不要通过微信等软件发送截图，会被压缩。

### 第二步：设置阵容（使用自动选人功能时需要）

1. 启动程序
2. 在「阵容设置」区域点击「编辑阵容」
3. 输入阵容名称、5个式神名字、阴阳师名字
4. 勾选「启用自动选人」
5. 可选：勾选「阵容轮换」设置轮换策略

> 式神名字必须与游戏内搜索一致，如"阿修罗"、"须佐之男"等

### 第三步：运行程序

#### 方式一：直接运行（需Python环境）

1. 安装Python 3.8+
2. 安装依赖：
   ```
   pip install -r requirements.txt
   ```
3. 将模板图片放入 `templates` 文件夹
4. 运行 `run.bat` 或执行 `python main.py`

#### 方式二：打包成exe（推荐）

1. 确保已安装依赖
2. 将模板图片放入 `templates` 文件夹
3. 双击运行 `build.bat`
4. 打包完成后，将 `templates` 文件夹复制到 `dist` 目录
5. 运行 `dist/阴阳师斗技自动挂机.exe`

### 第四步：开始使用

1. 启动阴阳师，进入斗技匹配界面
2. 启动本程序，点击"检测游戏窗口"确认已找到游戏
3. 设置挂机选项（最多场次、最长时间、是否自动选人等）
4. 点击"开始挂机"（或按 F6）
5. 程序会自动循环：匹配 → 选人 → 自动战斗 → 跳过结算 → 继续匹配

## 注意事项

1. **游戏窗口**：窗口不能被最小化，可以被遮挡但建议保持可见
2. **分辨率**：模板图片的分辨率需要与游戏运行时的分辨率一致。如果更换了分辨率，需要重新截图
3. **识别精度**：如果识别不准，可以在界面上拖动「识别灵敏度」滑块调整（默认0.8，范围0-1，值越低越宽松）
4. **自动选人**：自动选人功能依赖搜索功能，确保式神名字输入正确；搜索框位置和上滑距离基于默认分辨率，如有偏差请在代码中调整
5. **风险提示**：使用本程序有被封号的风险，请谨慎使用

## 常见问题

**Q: 提示找不到游戏窗口？**
A: 先启动阴阳师并进入游戏。程序会通过窗口标题和进程名双重校验查找游戏。

**Q: 按钮识别不准确/不点击？**
A: 请重新截取模板图片。确保截图只包含按钮本身，分辨率与游戏一致。也可以降低「识别灵敏度」试试。

**Q: 选人阶段不点击"确定"？**
A: 请检查 `templates/confirm.png` 是否正确截取了选人界面的"确定"按钮。

**Q: 战斗结束后卡住？**
A: 结算界面会自动点击屏幕中央，如果卡住可能是分辨率变化导致游戏窗口大小改变，建议重新截取模板。

**Q: 自动选人不准/选不上？**
A: 自动选人通过搜索+上滑实现，需要确保：
1. `search_button.png` 模板正确
2. 式神名字输入正确（与游戏内搜索一致）
3. 游戏分辨率与模板分辨率一致
4. 如果上滑距离不对，可调整 `selection_handler.py` 中的 `drag_target_y_ratio`
